## How To Install My App

How to install my app 

1) git clone https://github.com/Purple227/bellewise

2) cd into clone folder

3) sudo composer install

4) php artisan key:generate

5)  npm install

6) set up your respective database

7) php artisan migrate

7) php artisan serve
